# Code of Conduct

## Our Commitment

THEOS is a research project built on a single principle: **truth over inflation**.
We extend that principle to how we treat each other.

Contributors, maintainers, and users of this project are expected to participate
in a way that is honest, respectful, and constructive — whether in code reviews,
GitHub Discussions, issue comments, or any other project space.

---

## Standards

**We expect:**
- Honest reporting of experimental results — including negative results
- Respectful disagreement — challenge ideas, not people
- Accurate attribution of work and prior art
- Clear communication when something is uncertain or untested

**We do not accept:**
- Fabricated benchmarks or unverifiable performance claims
- Harassment, personal attacks, or dismissive behavior
- Deliberate misrepresentation of THEOS's capabilities
- Consciousness or sentience claims presented as scientific fact

---

## Scope

This code of conduct applies in all project spaces: GitHub Issues, Pull Requests,
Discussions, and any communication done on behalf of the project.

---

## Enforcement

Violations may be addressed by removing content, closing issues or PRs, or
blocking contributors from the repository. Serious or repeated violations will
be escalated appropriately.

To report an issue, open a private GitHub issue or contact the maintainer directly
via the repository's contact information.
